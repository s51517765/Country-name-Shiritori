# 国名しりとり

## 概要

共和国、合衆国、連邦などは基本的に除く

濁点、半濁点はどちらでもよい。「ド」でも「ト」でも可

最後は「ン」で終わらなくてもよい

使える国名は以下のリポジトリの'countries.dic.txt'を使用

https://github.com/jj1lis/myMisc/tree/master/shiritori
